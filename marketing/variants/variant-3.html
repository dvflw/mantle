Core Capabilities

Built for Serious Automation

Everything you need to orchestrate AI-powered workflows at scale, without vendor lock-in or hidden costs.

🔑

Bring Your Own Keys

Use your own API keys for OpenAI, Anthropic, and any OpenAI-compatible provider. No proxy fees, no data routing through third parties. Your keys, your data, your control.

📜

Infrastructure as Code

Define workflows in YAML. Version them in Git. Deploy with the same validate, plan, apply lifecycle you already know from Terraform.

🔄

Checkpoint & Resume

Every step checkpoints to Postgres. If your process crashes, it resumes from the last completed step. No lost work, no duplicate side effects.

🔒

Secrets as Opaque Handles

Credentials are AES-256-GCM encrypted and resolved at invocation time. They are never exposed as raw values in expressions or logs. Zero leakage by design.

📦

Single Binary

One Go binary. No runtime dependencies beyond Postgres. Deploy on bare metal, Kubernetes, or your laptop. It just works.

🔌

gRPC Plugin System

Extend Mantle with custom connectors via gRPC over subprocess. Write plugins in any language. Ship them alongside your workflows.

Under the Hood

Clean Architecture, Zero Bloat

A stack designed for operators, not just developers

Mantle is built in Go for predictable performance and minimal resource usage. No JVM warm-up, no Node.js event loop surprises. Just a static binary that starts in milliseconds and processes workflows deterministically.

  • CEL expressions for dynamic step configuration
  • Postgres-backed state for durability and queryability
  • HashiCorp go-plugin protocol for safe extension
  • Helm chart included for Kubernetes deployments
  • Full audit trail from day one
