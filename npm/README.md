# @dvflw/mantle

npm wrapper for the [Mantle CLI](https://github.com/dvflw/mantle) — headless AI workflow automation.

## Usage

Run without installing:

```sh
npx @dvflw/mantle version
npx @dvflw/mantle validate workflow.yaml
npx @dvflw/mantle run my-workflow
```

Or install globally:

```sh
npm install -g @dvflw/mantle
mantle version
```

## Supported platforms

- linux (x64, arm64)
- macOS (x64, arm64)

## About

Mantle is an open-source, headless AI workflow automation platform. BYOK, IaC-first, enterprise-grade.

For full documentation, see the [main repository](https://github.com/dvflw/mantle).
